address-of演算子 (&) とはどういう意味ですか?

著者 ダニエル・ゴンザレス

ほとんどのプログラミング言語では、address-of演算子&を使用して、変数のメモリ アドレスを取得します。例えば、xという値を含む変数5がある場合、その前にx演算子を使用することで、&のメモリ アドレスを取得できます。したがって、&xと書くと、5という値がメモリに格納されているメモリ アドレスが得られます。

詳しくは

address-of演算子&は、プログラミングで 2 つの主な目的で一般的に使用されます。

まず、前述のように&演算子を使用して変数のメモリ アドレスを取得します。これは、変数のアドレスを引数として関数に渡したい場合など、特定の状況で役立ちます。

第二に、&演算子は、変数への参照を作成するために逆参照演算子*と組み合わせて使用​​されることもよくあります。この場合、&演算子は変数のアドレスを取得するために使用され、*演算子はその変数への参照を作成するために使用されます。

この参照は、ポインタのように、元の変数にアクセスして変更するために使用できます。

たとえば、C++ では、次のように&および*演算子を使用できます。

int x = 5;
int *ptr = &x;  // Get the address of x and create a reference to it using the * operator
*ptr = 10;      // Use the reference to modify the value of x

// At this point, the value of x is 10

この例では、ptr変数はx変数への参照です。つまり、ptrを使用してxの値にポインタのようにアクセスして変更できることを意味します。

&演算子は、プログラミングでメモリアドレスや参照を操作するのに便利なツールであり、さまざまな状況で多くの用途があります。
