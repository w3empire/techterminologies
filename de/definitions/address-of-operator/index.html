Was bedeutet der 'Address-of'-Operator (&)?

autor Daniel Gonzales

In den meisten Programmiersprachen wird der address-of-Operator & verwendet, um die Speicheradresse einer Variablen zu erhalten. Wenn Sie beispielsweise eine Variable x haben, die den Wert 5 enthält, können Sie die Speicheradresse von x erhalten, indem Sie den Operator & davor verwenden. Wenn Sie also &x schreiben, erhalten Sie die Speicheradresse, an der der Wert 5 gespeichert ist.

Mehr Informationen

Der address-of-Operator & wird üblicherweise für zwei Hauptzwecke in der Programmierung verwendet.

Zunächst wird, wie bereits erwähnt, der Operator & verwendet, um die Speicheradresse einer Variablen zu erhalten. Dies kann in bestimmten Situationen nützlich sein, z. B. wenn Sie die Adresse einer Variablen als Argument an eine Funktion übergeben möchten.

Zweitens wird der Operator & auch oft in Kombination mit dem Dereferenzierungsoperator * verwendet, um eine Referenz auf eine Variable zu erstellen. In diesem Fall wird der Operator & verwendet, um die Adresse einer Variablen zu erhalten, und der Operator * wird verwendet, um eine Referenz auf diese Variable zu erstellen. Diese Referenz kann dann verwendet werden, um auf die ursprüngliche Variable zuzugreifen und sie zu ändern, als wäre sie ein Zeiger.

In C++ könnten Sie beispielsweise die Operatoren & und * wie folgt verwenden:

int x = 5;
int *ptr = &x;  // Get the address of x and create a reference to it using the * operator
*ptr = 10;      // Use the reference to modify the value of x

// At this point, the value of x is 10

In diesem Beispiel ist die Variable ptr ein Verweis auf die Variable x, was bedeutet, dass Sie ptr verwenden können, um auf den Wert von x zuzugreifen und ihn zu ändern, als wäre es ein Zeiger.

Der Operator & ist ein nützliches Werkzeug für die Arbeit mit Speicheradressen und Referenzen bei der Programmierung und hat viele Anwendungen in einer Vielzahl von Situationen.
